## Why this isn't implemented yet

This content describes the real official Goethe Start Deutsch 1 Sprechen
structure (fixed Teil 1 self-intro fields, fixed Teil 2 keyword cards,
fixed Teil 3 action cards, scored 3+6+6=15 points). DeutschWeg's actual
live Sprechen engine (`server.js` `POST /api/sprechen/topics/generate`,
`sprechen.html`) is structurally different:

- **Content**: 3 dynamic parts (Sich vorstellen / Bild beschreiben /
  Gemeinsam planen), freshly generated by Claude every session — there is
  no fixed-card data model in the schema or the UI today.
- **Linkage**: Sprechen is never tied to a specific mock exam set —
  `mock_exam_sets.sprechen_topic_ids` stays `null` on every set by design
  (see `uebungssatz-2-gold-standard.md` §5). Every A1 set (1, 2, and now
  3) gets the identical dynamic engine.
- **Scoring**: `exam-vault.html`'s combined-score code
  (`computeAndShowMockExamCombinedScore`) deliberately never folds
  Sprechen into the learner's /100 — it's a documented, reviewed decision
  ("deliberately NEVER folded into this number — reviewed decision, not
  an oversight"), shown instead as a separate, non-scored "Sprechen
  readiness" block.

Building the fixed-card structure this content describes would mean a new
data model and new scoring wiring that touches the shared combined-score
code (used by Mock Exam 1 and 2 too) — a real feature decision, not
something to improvise while authoring content. Flagged, not built,
pending a separate decision on if/how to wire it in.

## Teil 1 — Sich vorstellen (3 Punkte)

Felder: Name, Alter, Land, Wohnort, Sprachen, Beruf
Zusatzfunktionen: ein Wort/den Wohnort buchstabieren; eine Nummer nennen

## Teil 2 — Um Informationen bitten und Informationen geben (6 Punkte)

Thema: "Unterwegs"

1. **Bahnhof** — Wo ist der Bahnhof? / Wie komme ich zum Bahnhof? / Ist der Bahnhof weit? / Welche Linie fährt zum Bahnhof?
2. **Adresse** — Wie ist Ihre Adresse? / In welcher Straße wohnen Sie? / Wie ist Ihre Hausnummer?
3. **Bus** — Welche Linie fährt zum Zentrum? / Wo ist die Bushaltestelle? / Wie oft fährt die Linie?
4. **Apotheke** — Wo ist eine Apotheke? / Hat die Apotheke heute geöffnet? / Wie komme ich zur Apotheke?
5. **Eingang** — Wo ist der Eingang? / Ist das der Haupteingang? / Welchen Eingang soll ich nehmen? / Ist der Eingang vorne oder hinten?
6. **Treffpunkt** — Wo treffen wir uns? / Um wie viel Uhr treffen wir uns? / Treffen wir uns am Eingang? / Ist der Treffpunkt vor dem Bahnhof?

## Teil 3 — Bitten formulieren und darauf reagieren (6 Punkte)

1. **Weg auf einer Karte zeigen** — Zeigen Sie mir bitte den Weg. / Können Sie mir das auf der Karte zeigen?
2. **Tür öffnen** — Öffnen Sie bitte die Tür. / Können Sie mir bitte die Tür öffnen? — negative Reaktion: "Tut mir leid, ich habe keinen Schlüssel."
3. **Platz machen** — Machen Sie bitte Platz. / Können Sie bitte etwas Platz machen? — alt.: Lassen Sie mich bitte durch.
4. **Etwas wiederholen** — Können Sie das bitte wiederholen? / Sagen Sie das bitte noch einmal.
5. **Aufzugknopf drücken** — Drücken Sie bitte die Drei. / Können Sie bitte den Knopf für den dritten Stock drücken?
6. **Adresse aufschreiben** — Schreiben Sie mir bitte die Adresse auf. / Können Sie die Adresse bitte aufschreiben?

## Image-asset brief (for when/if this gets built — not used today)

**Global rule (Teil 2):** written keyword remains dominant; icon secondary; no narrative scenes.

**Teil 2:**
1. Bahnhof — station icon.
2. Adresse — neutral house/street sign, no field emphasized.
3. Bus — exterior bus icon.
4. Apotheke — pharmacy cross icon.
5. Eingang — one neutral entrance icon; avoid multiple doors, arrows, pointing figures, or narrative action scenes.
6. Treffpunkt — two figures + landmark; clock small/secondary only.

**Teil 3:**
1. Map-showing — two figures, helper not yet pointing (action must not appear completed).
2. Tür öffnen — closed door, full-handed figure, helper nearby.
3. Platz machen — suitcase only (not pram, not both), blocked aisle, one person needing to pass, another occupying the space.
4. Wiederholen — hand-to-ear + speech bubble showing speech was not understood, not a question mark alone.
5. Aufzugknopf drücken — lift interior; person with full hands NOT beside buttons; helper positioned beside buttons; floor 3 clearly visible; no button already illuminated.
6. Adresse aufschreiben — empty notepad + pen (visibly still empty); phone shows only neutral placeholder text or a fictional, partly unreadable address — no real personal information.
