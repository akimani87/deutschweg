Key collocations in politics/society: Maßnahmen ergreifen (to take measures), ein Gesetz verabschieden (to pass a law), eine Debatte auslösen (to trigger a debate), einen Beitrag leisten (to make a contribution), Integration fördern (to promote integration). These appear constantly in B2 texts.

Kollokationen

Kollokationen sind feste Wort-Kombinationen — kein einzelnes Wort ist falsch, aber die Kombination klingt unnatürlich ohne das richtige Partnerwort.

VERBNOMENKORREKT
ergreifenMaßnahmenMaßnahmen ergreifen ✓
leistenBeitrag / Widerstandeinen Beitrag leisten ✓
stellenFrage / Antrageine Frage stellen ✓
vorantreibenDigitalisierung / Reformdie Reform vorantreiben ✓
gewährleistenSicherheit / DatenschutzSicherheit gewährleisten ✓
auslösenDebatte / Reaktioneine Debatte auslösen ✓
berücksichtigenInteressen / AspekteInteressen berücksichtigen ✓
❌ Maßnahmen machen — falsch!
✓ Maßnahmen ergreifen / treffen
🇬🇧 English safety net

Collocations: Fixed verb-noun pairs in German that you must memorize as units. The most common mistakes: using machen for everything (❌ Maßnahmen machen → ✓ Maßnahmen ergreifen/treffen), or translating directly from English. Learn: Beitrag leisten, Antrag stellen, Widerstand leisten, Frage aufwerfen, Interesse wecken.

Register: formell vs. umgangssprachlich

In der B2-Schreibprüfung erwartet man formelles Deutsch. Wer umgangssprachliche Wörter benutzt, verliert Punkte für Stil.

FORMELL (B2)UMGANGSSPRACHLICH
belegen / nachweisenzeigen
erörtern / darlegenbesprechen / erklären
aufgrund / hinsichtlichwegen / über
in Betracht ziehendenken an
sich auf etwas beziehenüber etwas reden
die Inanspruchnahmedie Nutzung
infolgedessendeshalb
im Hinblick aufbezüglich
Merke! Konjunktionen des Registers:
• Formal: jedoch, dennoch, gleichwohl, nichtsdestotrotz
• Neutral: aber, trotzdem, trotz
• Umgangssprachlich: bloß, halt, eben
🇬🇧 English safety net

Register in German: B2 writing tasks (formal letter, essay) require formal register. Swap: zeigen → belegen, wegen → aufgrund, besprechen → erörtern, deshalb → infolgedessen/daher. Connectors: use jedoch/dennoch instead of aber/trotzdem for formal writing. Avoid modal particles (ja, doch, halt, eben) in formal writing.

Wortbildung meistern

Deutsche Wortbildung ist systematisch. Wer Präfixe, Suffixe und Komposita versteht, kann Tausende neuer Wörter erschließen.

PRÄFIXE UND BEDEUTUNG

✅ Komposita dekodieren: letztes Wort = Grundwort (Bedeutung + Genus). Erstes Wort = Bestimmungswort (Modifikation).
Beispiel: Klimaschutzmaßnahme = Maßnahme (die, fem.) + zum Klimaschutz
🇬🇧 English safety net

Word formation: German builds new words systematically. Prefixes: ver- (change/error), be- (makes intransitive verbs transitive), ent- (removal/reversal), er- (achievement/start), un- (negation). Suffixes: -ung (noun, feminine), -heit/-keit (abstract noun, feminine), -lich (adjective). Compound words: the last element determines gender and core meaning.
