Erweitertes Passiv

Das Passiv auf B2-Niveau geht weit über das einfache Vorgangspassiv hinaus. Du lernst das Zustandspassiv, Passivumschreibungen (sein+zu, lassen+sich), modales Passiv und alle Zeitformen — unverzichtbar für akademische und formelle Texte.

🇬🇧 What will I learn here?

At B2, you go beyond basic passive (wird gemacht) to master: Zustandspassiv (ist gemacht), passive with modal verbs (muss gemacht werden), passive alternatives (ist zu machen, lässt sich machen), and all tenses of the passive.

Passive voice is essential for academic writing, formal reports, and B2 reading texts — it shifts focus from the actor to the action or result.

Vorgangs- vs. Zustandspassiv

Der entscheidende Unterschied: Das Vorgangspassiv beschreibt den Prozess (etwas passiert). Das Zustandspassiv beschreibt das Ergebnis (etwas ist fertig/geschehen).

🇬🇧 Two types of passive explained

Vorgangspassiv (process passive): werden + Partizip II. Focus on the action happening. "The door is being opened." Das Fenster wird geöffnet.

Zustandspassiv (state passive): sein + Partizip II. Focus on the resulting state. "The door is open (= has been opened)." Das Fenster ist geöffnet.

Tip: Zustandspassiv often looks like an adjective. The key test: can you add "schon/bereits" (already)? If yes → Zustandspassiv.

TypBildungBedeutungBeispiel
Vorgangwerden + P.IIProzess läuftwird gebaut
Zustandsein + P.IIErgebnisist gebaut
VORGANG
Das Haus wird gebaut. (jetzt, Prozess)
Vorgangspassiv: werden + Partizip II. Etwas geschieht gerade.
ZUSTAND
Das Haus ist gebaut. (fertig, Ergebnis)
Zustandspassiv: sein + Partizip II. Ergebnis eines abgeschlossenen Vorgangs.
VORGANG
Die Tür wird geöffnet. (jemand öffnet sie gerade)
ZUSTAND
Die Tür ist geöffnet. (sie steht offen)
Merke! „Das Buch ist geschrieben" kann auch Adjektiv sein (wie „Das Buch ist interessant"). Der Kontext entscheidet. Test: Kann man „von jemandem" hinzufügen? → Zustandspassiv.
Weitere Beispiele (B2-Kontext)
Kofi fragte, ob der Vertrag schon unterzeichnet sei.
Zustandspassiv im KI (indirekte Rede): sein → sei
Amina berichtete, die Daten würden gerade ausgewertet.
Vorgangspassiv Gegenwart im KI: werden → würden (wir/sie → KII)

Alle Zeitformen des Passivs

Das Passiv existiert in allen Zeitformen. Auf B2-Niveau musst du besonders Passiv Perfekt und Plusquamperfekt sicher bilden können — sie erscheinen häufig in Berichten und Analysen.

🇬🇧 Passive tenses overview

Passive in all tenses uses werden conjugated in that tense + Partizip II of the main verb.

Perfekt passive is tricky: it's "ist worden" not "ist geworden". The "ge-" is dropped from "geworden" → "worden".

Plusquamperfekt: war + worden. Futur: wird + werden (sounds strange, rarely used — prefer active).

ZeitformBildung (er/es)Beispiel
Präsenswird + P.IIwird gebaut
Präteritumwurde + P.IIwurde gebaut
Perfektist + P.II + wordenist gebaut worden
Plusquamperf.war + P.II + wordenwar gebaut worden
Futur Iwird + P.II + werdenwird gebaut werden
Achtung! Passiv Perfekt: „ist … worden" — NICHT „ist … geworden". Das ist einer der häufigsten B2-Fehler!

❌ Das Haus ist gebaut geworden.
✓ Das Haus ist gebaut worden.
Ngozi berichtete: Der Bericht wurde gestern fertiggestellt.
Passiv Präteritum — häufig in Berichten und Erzählungen
Die neue Brücke ist 2023 eröffnet worden.
Passiv Perfekt — ist + P.II + worden
Als wir ankamen, war das Gebäude bereits abgerissen worden.
Passiv Plusquamperfekt — war + P.II + worden

Modales Passiv

Modalverben und Passiv kombiniert entstehen sehr häufig in formellen Texten: Gesetzen, Verordnungen, wissenschaftlichen Empfehlungen. Wer diese Konstruktion beherrscht, liest authentische B2-Texte mühelos.

🇬🇧 Modal + passive explained

Combine a modal verb with passive: Modalverb + Partizip II + werden

Präsens: Das muss gemacht werden. (This must be done.)

Präteritum: Das musste gemacht werden. (This had to be done.)

Perfekt: Das hat gemacht werden müssen. (This had to be done.) — rare but appears in formal texts.

MODALES PASSIV — FORMEL
Modalverb + Partizip II + werden
Das Problem muss gelöst werden.

müssen + eingereicht + werden — typisch für offizielle Anforderungen
Die Ergebnisse können nicht überprüft werden.
können + überprüft + werden — Möglichkeit verneint
Chidi erklärte, das Protokoll solle unverzüglich veröffentlicht werden.
Modales Passiv + KI (indirekte Rede): soll → solle
Diese Regelung darf nicht umgangen werden.
Verbot: dürfen (negiert) + P.II + werden

Passivumschreibungen

Deutsch hat elegante Alternativen zum echten Passiv — besonders in formeller und wissenschaftlicher Sprache. Diese Umschreibungen klingen natürlicher und erscheinen häufig in B2-Texten.

🇬🇧 Passive alternatives explained

sein + zu + Infinitiv: expresses necessity/possibility (equivalent to "must/can be done"). Das ist zu beachten = This must be noted.

lassen + sich + Infinitiv: expresses possibility (equivalent to "can be done"). Das lässt sich lösen = That can be solved.

man + Aktiv: informal alternative. Man muss das beachten = This must be noted.

KonstruktionBedeutungBeispiel

lassen + sich + Inf.kann gemacht werdenDas lässt sich lösen.
sich + lassen + Inf.= lassen+sichDer Fehler lässt sich korrigieren.
Adjektiv auf -bar/-lichkann gemacht werdenDas ist lösbar / verständlich.

Dieser Punkt ist besonders zu beachten.
= muss beachtet werden (obligatorisch/empfohlen)
LASSEN+SICH

= kann nicht ignoriert werden (Möglichkeit)

= kann durchgeführt werden — Adjektiv-Ableitung auf -bar
Tipp für B2-Schreiben: Wechsle zwischen echtem Passiv und Umschreibungen ab — das zeigt Stilkompetenz und macht deine Texte abwechslungsreicher.

Aktiv ↔ Passiv & B2-Strategien

Beim Umformen von Aktiv in Passiv und zurück müssen Kasus, das Agens (von + Dativ) und der Satzaufbau beachtet werden. Passiv ohne Agens ist besonders in formellen Texten häufig.

🇬🇧 Active ↔ Passive transformation

Active → Passive: Object becomes subject (Nominative). Subject becomes agent with "von + Dative" (optional). Verb becomes werden + Partizip II.

Passive without agent: extremely common in formal German — the actor is unknown, unimportant, or obvious from context. "Es wird diskutiert" (there is discussion going on).

Aktiv → Passiv: Schritt für Schritt
AKTIV
Fatima liest den Bericht.
↓ Objekt wird Subjekt | Subjekt → von + Dat. | Verb → wird + P.II
PASSIV
Der Bericht wird (von Fatima) gelesen.
Passiv ohne Agens (häufig!)
Über dieses Thema wird viel diskutiert.
Kein Agens — der Handelnde ist allgemein/unbekannt. Typisch für formelle Texte.
